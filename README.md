# telegram-bot-chat

## 前言

- 基于Vue3+Vite+TypeScript开发的一个纯开源免费版的Telegram Bot聊天与管理的H5平台

### 安装
```shell
git clone https://github.com/Noi-q/telegram-bot-chat.git
cd telegram-bot-chat
yarn install
```

### 运行

```shell 
yarn dev --host` 或 `yarn dev
```
